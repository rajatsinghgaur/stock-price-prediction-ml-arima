# 📊 Stock Price Prediction using Machine Learning & ARIMA

##  Project Overview
This project analyzes 26 years of historical stock data of Reliance Industries Limited (2000–2026) to build predictive models using Machine Learning and ARIMA Time-Series Forecasting.

##  Key Features
- Data Cleaning & Preprocessing
- Technical Indicators (MA, RSI, MACD)
- Feature Engineering
- Machine Learning Model Comparison
- ARIMA Forecast (March 2026)
- Risk Analysis (VaR, Volatility, Drawdown)
- Trading Strategy Backtesting

##  Key Results
- Annualized Return: 34.31%
- Annualized Volatility: 75.58%
- March 2026 Forecast: ₹1395.85

##  Tools Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Statsmodels
- Matplotlib

##  Conclusion
The project demonstrates integration of financial analytics, machine learning, and time-series modeling for structured stock forecasting.
